مرجع کامل C#

C# چیست؟

C# زبان رسمی مایکروسافت برای توسعه ویندوز، وب، بازی‌سازی (Unity) و بک‌اند با ASP.NET Core است.

کاربردهای C#

  • توسعه بازی با Unity
  • ساخت وب‌سرویس با ASP.NET Core
  • برنامه‌نویسی ویندوز (WPF – WinForms)
  • سیستم‌های سازمانی

فریم‌ورک‌ها

  • ASP.NET Core
  • Unity
  • Entity Framework Core
