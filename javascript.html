JavaScript چیست؟

JavaScript زبان اصلی وب است و برای فرانت‌اند، بک‌اند (Node.js)، اپ موبایل، دسکتاپ و حتی هوش مصنوعی استفاده می‌شود.

کاربردهای JavaScript

  • Front-end با React ،Vue ،Angular
  • Back-end با Node.js
  • ساخت اپ موبایل (React Native)
  • اپ دسکتاپ (Electron)

فریم‌ورک‌ها و کتابخانه‌ها

  • React
  • Vue
  • Angular
  • Express.js
